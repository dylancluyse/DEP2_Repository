# DEP2_Repository



## Package manager

We use Poetry for version managing. To run the code locally, run:

```python
$ poetry install
$ poetry shell
```

## DB access

Please see the `.env.example` file and create a `.env` file. (Ask for the missing environment variables)

## PostgreSQL Queries
